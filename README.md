# 概要
Ruby on ralisの勉強として作成したポートフォリオです。
ユーザーの商品や食べ物を写真と共に投稿し、閲覧者に紹介、おすすめするアプリケーションです。

## バージョン
* ruby '2.6.5'
* rails '6.0.0'
